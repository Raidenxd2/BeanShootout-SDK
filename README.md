# The Great Bean Shootout SDK
Unity version: 6000.3.8f1

# Installation
Create a project with the 3D (Built-In Render Pipeline) template \
Open the Package Manager (Window/Package Management/Package Manager) \
Click the plus on the top right and click install package from Git URL... \
Then put https://github.com/Raidenxd2/com.unity.inputsystem.git and click Install \
If a dialog comes up about the Input System, click no \
Do the same thing but with https://github.com/Raidenxd2/BeanShootout-SDK.git instead \
You will recieve a few messages, read them all and click yes if you agree for changes to be made to the project
